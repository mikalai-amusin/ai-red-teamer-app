# 💀 DevRoast: The AI Red-Teamer for Indie Hackers

**Market Validation through Brutal Analysis.**

Solo founders often live in echo chambers. **DevRoast** is a specialized AI agent designed to act as your most cynical venture capitalist partner. It performs "Red-Teaming" on your startup pitch, exposing market flaws, identifying competitor threats, and suggesting tactical pivots—all in under 30 seconds.

---

## 🚀 Experience the Brutality
**[Live App: ai-red-teamer.vercel.app](https://ai-red-teamer.vercel.app)**

---

## 🔥 Key Features
*   **Brutal Reality Checks:** Get a ruthlessly analytical report on your startup's structural weaknesses.
*   **Cinematic Terminal UI:** Experience a high-tension loading sequence that builds anticipation for the roast.
*   **YC-Trained Logic:** The underlying prompt is engineered to simulate the feedback of high-stakes accelerator partners.
*   **Export to Markdown:** Instant download of your report for internal "Founder Therapy" or tactical planning.
*   **Vercel Analytics:** Native integration for tracking visitor engagement during high-traffic launches.

## 🛠️ Tech Stack
*   **Framework:** Next.js (App Router)
*   **Language:** TypeScript
*   **Styling:** Vanilla CSS (Bespoke Modules & Global Variables)
*   **AI Engine:** Google Gemini SDK (`@google/genai`)
*   **Analytics:** Vercel Analytics
*   **Animations:** Framer Motion

## ⚙️ Local Development

### 1. Prerequisites
*   Node.js 20+
*   A Google Gemini API Key (get one at [aistudio.google.com](https://aistudio.google.com/))

### 2. Setup
```bash
# Clone the repository
git clone https://github.com/mikalai-amusin/ai-red-teamer-app.git

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env.local
# Add your GEMINI_API_KEY to .env.local
```

### 3. Run
```bash
npm run dev
```
